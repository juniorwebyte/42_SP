Exercício 05: build.sh

Objetivo

Criar um script chamado build.sh que cria pastas com base nos argumentos passados para o script, adicionando o prefixo ex ao nome da pasta. Se nenhum argumento for passado, o script exibe a mensagem No arguments supplied.

Passo a Passo

  1. Navegar até a pasta ex05:

    Primeiramente, vá até a pasta onde o exercício será realizado. No caso, a pasta ex05 dentro da estrutura de pastas já criada. Se ela não existir, crie-a utilizando os seguintes comandos:

    cd ~/missao_42/cell00
mkdir ex05
cd ex05
  2. Criar o script build.sh com o código necessário:

    Agora, crie o arquivo build.sh com o seguinte conteúdo:

    #!/bin/bash

# Verifica se foram passados argumentos para o script
if [ $# -eq 0 ]; then
    echo "No arguments supplied"
else
    # Para cada argumento, cria o diretório com o prefixo "ex"
    for arg in "$@"; do
        mkdir "ex$arg"
    done
fi
  3. Verificar o conteúdo do script:

    Depois de criar o script, você pode verificar se o conteúdo foi inserido corretamente utilizando o comando:

    cat build.sh

    O conteúdo do arquivo deve ser:

    #!/bin/bash

# Verifica se foram passados argumentos para o script
if [ $# -eq 0 ]; then
    echo "No arguments supplied"
else
    # Para cada argumento, cria o diretório com o prefixo "ex"
    for arg in "$@"; do
        mkdir "ex$arg"
    done
fi
  4. Tornar o script executável:

    Para que o script possa ser executado, é necessário torná-lo executável com o seguinte comando:

    chmod +x build.sh
  5. Testar o script:

    Agora, execute o script com diferentes números de argumentos e verifique se as pastas são criadas corretamente. Aqui estão alguns exemplos:

      • No arguments supplied
