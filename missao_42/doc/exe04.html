Exercício 04: argv.sh

Objetivo

Escrever um script chamado argv.sh que exibe até 3 argumentos passados a este script. Se nenhum argumento for passado, o script deve mostrar uma mensagem informando isso.

Passo a Passo

  1. Criar o arquivo argv.sh:

    Primeiro, crie o arquivo argv.sh dentro da pasta ex04 com o seguinte conteúdo:

    #!/bin/bash

# Verifica se nenhum argumento foi passado
if [ $# -eq 0 ]; then
    echo "No arguments supplied"
else
    count=0
    # Loop para percorrer os argumentos
    for arg in "$@"; do
        # Verifica se o número de argumentos não excede 3
        if [ $count -lt 3 ]; then
            echo "$arg"
            ((count++))
        fi
    done
fi

    Este código realiza a verificação dos argumentos passados ao script. A variável especial $# conta o número de argumentos, e o script imprime até três argumentos passados.

  2. Tornar o arquivo executável:

    Para poder executar o script diretamente, torne-o executável com o comando:

    chmod +x argv.sh

    Este comando atribui a permissão de execução ao script, permitindo que você execute-o diretamente.

  3. Executar o script sem argumentos:

    Execute o script sem argumentos:

    ./argv.sh

    A saída esperada será:

    No arguments supplied

    Quando nenhum argumento é passado, o script verifica isso e exibe a mensagem indicando que nenhum argumento foi fornecido.

  4. Testar com um argumento:

    Execute o script passando um único argumento:

    ./argv.sh 42

    A saída esperada será:

    42

    Neste caso, o script exibe o único argumento passado: 42.

  5. Testar com múltiplos argumentos:

    Execute o script com mais de 3 argumentos:

    ./argv.sh 2 40 42

    A saída esperada será:

    2
40
42

    Quando mais de três argumentos são passados, o script exibe até três deles, conforme a lógica implementada no laço for.
