Exercício 01: Z

Objetivo

Para este exercício, o objetivo é criar um arquivo chamado z que, quando utilizado com o comando cat, mostre a letra "Z" seguida de uma nova linha.

Passo a Passo

  1. Navegar até a pasta correta:

    Primeiramente, vá até a pasta ex01, onde você deve criar o arquivo. Se você já estiver na pasta missao_42/cell00/ex00, basta navegar até cell00/ex01:

    cd ~/missao_42/cell00/ex01

    Se a pasta ex01 ainda não existir, crie-a:

    mkdir ex01
cd ex01
  2. Criar o arquivo z:

    Agora, crie o arquivo z utilizando o comando echo:

    echo "Z" > z

    Esse comando cria o arquivo z e escreve a letra "Z" nele, seguida de uma nova linha.

  3. Verificar o conteúdo do arquivo com cat:

    Após criar o arquivo, você pode verificar se ele está correto utilizando o comando cat:

    cat z

    A saída deve ser:

    Z
  4. Estrutura final:

    Após executar os passos, a pasta ex01 deve conter o arquivo z, e ao executar cat z, você verá a letra "Z" na tela.
