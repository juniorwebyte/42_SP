Exercício 00: O Básico

Objetivo

Organizar os exercícios criando uma estrutura de pastas específica. O exercício envolve criar as pastas necessárias para organizar os exercícios futuros.

Passo a Passo

  1. Criar a pasta missao_42 no diretório home e navegar até ela:

    Primeiramente, vá para o seu diretório home (normalmente, você já está nele quando abre o terminal). Em seguida, crie a pasta missao_42 e acesse-a:

    mkdir ~/missao_42
cd ~/missao_42
  2. Criar a pasta cell00 e navegar até ela:

    Dentro da pasta missao_42, crie a pasta cell00 e entre nela:

    mkdir cell00
cd cell00
  3. Criar a pasta ex00:

    Agora, dentro de cell00, crie a pasta ex00:

    mkdir ex00
  4. Finalizar a estrutura de pastas:

    Até aqui, você criou as pastas conforme solicitado. Agora, a estrutura de diretórios está da seguinte forma:

    ~/missao_42/cell00/ex00

    A partir de agora, todos os seus exercícios serão organizados dentro de pastas específicas como ex00, ex01, etc., conforme indicado nas instruções.
