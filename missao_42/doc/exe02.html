Exercício 02: midLS

Objetivo

Criar um arquivo chamado midLS que, quando executado, lista todos os arquivos e pastas da pasta atual (exceto os arquivos ocultos) e os separa por vírgulas.

Explicação: Este exercício tem como objetivo ensinar a criar um script simples no Linux que utilize comandos básicos do sistema para manipular arquivos e pastas, como o comando ls, que lista os arquivos e diretórios de um diretório, e o grep para excluir arquivos ocultos (arquivos que começam com ponto).

Passo a Passo

  1. Navegar até a pasta ex02:

    Primeiramente, você precisa estar na pasta onde o exercício será realizado. No caso, a pasta ex02 dentro da estrutura de pastas já criada. Se ela não existir, você deve criá-la utilizando os seguintes comandos:

    cd ~/missao_42/cell00
mkdir ex02
cd ex02
  2. Criar o arquivo midLS com o comando necessário:

    Agora, você deve criar o arquivo midLS, que vai conter o comando necessário para listar todos os arquivos e pastas da pasta atual, excluindo arquivos ocultos (aqueles que começam com ponto) e separando-os por vírgulas. Para isso, utilize o comando:

    echo "ls -1 | grep -v '^\.' | paste -sd, -" > midLS
  3. Verificar o conteúdo do arquivo:

    Depois de criar o arquivo midLS, você pode verificar se o conteúdo foi inserido corretamente com o comando:

    cat midLS

    O conteúdo do arquivo deve ser:

    ls -1 | grep -v '^\.' | paste -sd, -
  4. Tornar o script executável (opcional):

    Se quiser rodar o arquivo como um script, você precisa torná-lo executável. Para isso, utilize o comando:

    chmod +x midLS
  5. Testar o comando:

    Agora, para testar o script e ver se ele funciona corretamente, você pode executar o comando:

    ./midLS

    Explicação: O comando ./ executa o arquivo script no terminal. Ele deve listar todos os arquivos e pastas da pasta atual (exceto os ocultos) e separá-los por vírgulas.
