Exercício 03: count_files.sh

Objetivo

Criar um script chamado count_files.sh que contabiliza e exibe o número de arquivos regulares e diretórios presentes na pasta atual. O script não deve contar arquivos ocultos (aqueles que começam com ponto).

Passo a Passo

  1. Navegar até a pasta ex03:

    Primeiramente, vá até a pasta ex03, onde o arquivo será criado. Se essa pasta não existir, crie-a com os seguintes comandos:

    cd ~/missao_42/cell00
mkdir ex03
cd ex03
  2. Criar o arquivo count_files.sh com o comando necessário:

    Agora, crie o arquivo count_files.sh contendo o comando que contará e exibirá os arquivos e diretórios da pasta atual:

    echo "find . -type f | wc -l; find . -type d | wc -l" > count_files.sh
  3. Verificar o conteúdo do arquivo:

    Depois de criar o arquivo count_files.sh, verifique se o conteúdo foi inserido corretamente com o comando:

    cat count_files.sh

    O conteúdo do arquivo deve ser:

    find . -type f | wc -l; find . -type d | wc -l
  4. Tornar o script executável:

    Se quiser executar o script diretamente, torne-o executável com o seguinte comando:

    chmod +x count_files.sh
  5. Testar o script:

    Agora, você pode executar o script com o comando:

    ./count_files.sh

    O script retornará dois números: o primeiro é o número de arquivos regulares e o segundo é o número de diretórios na pasta atual.
