Documentação da Aplicação Investidores

Visão Geral

A aplicação "Investidores" é uma plataforma voltada para oferecer soluções de investimento, incluindo funcionalidades para gestão de afiliados, pagamentos e outras integrações financeiras. O sistema está estruturado dentro de uma arquitetura modular, permitindo fácil expansão e manutenção. A aplicação foi desenvolvida para ser integrada ao Laravel, utilizando boas práticas e tecnologias modernas.

Requisitos para Instalação

  • PHP 7.4+ ou versão superior
  • Composer instalado
  • Laravel 8.x ou superior
  • MySQL ou outro banco de dados suportado pelo Laravel

Passo a Passo para Instalação

1. Clonando o Repositório

Clone o repositório da aplicação para sua máquina local.

git clone https://github.com/seu-usuario/investidores.git
cd investidores

2. Instalando as Dependências

Instale as dependências do Laravel usando o Composer.

composer install

3. Configurando o Ambiente

Copie o arquivo `.env.example` para `.env` e configure as variáveis de ambiente, como o banco de dados.

cp .env.example .env

4. Gerando a Chave da Aplicação

Gere a chave de criptografia para a aplicação.

php artisan key:generate

5. Executando as Migrações

Execute as migrações do banco de dados para criar as tabelas necessárias.

php artisan migrate

6. Servindo a Aplicação

Inicie o servidor de desenvolvimento do Laravel.

php artisan serve

A aplicação estará disponível em http://localhost:8000.

Manipulando a Estrutura da Aplicação

Arquivos Principais

Na estrutura de pastas, você encontrará os seguintes arquivos principais:

  • index.php: Página principal da aplicação.
  • vendiaqui.php: Página voltada para a plataforma de vendas.
  • webytemusic.php: Página voltada para a plataforma de música Webyte Music.

1. Adicionando Novos Arquivos de Estilo

Para adicionar novos estilos à aplicação, basta incluir o arquivo CSS dentro da pasta /assets/css/ e referenciá-lo no arquivo index.php ou em qualquer página que deseje estilizar.

2. Adicionando Novas Imagens

Para adicionar novas imagens, coloque-as dentro da pasta /assets/img/ e utilize o caminho relativo nas páginas HTML ou PHP.

3. Modificando Funcionalidades PHP

Arquivos PHP reutilizáveis estão localizados na pasta /includes/. Se precisar modificar ou adicionar novos formulários, edite os arquivos existentes ou crie novos scripts PHP na pasta /includes/.

Conclusão

Essa documentação fornece uma visão geral sobre como configurar e manipular a aplicação "Investidores" dentro de um ambiente Laravel. A estrutura modular permite fácil expansão e personalização de funcionalidades conforme as necessidades do projeto.
